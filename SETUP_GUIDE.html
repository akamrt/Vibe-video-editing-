Setup Guide — Get running in 2 minutes

⚡ The Short Version

If you just want to get going, double-click this file:

START.bat (Windows)  or  ./START.sh (Mac)

It will automatically install everything needed on first run.

Windows Setup

1

📁Get the Files

Download or copy the VibeCut AI folder to your computer. It doesn't matter where — your Desktop, Documents, or Downloads folder all work fine.

2

📦Install Everything

Open the VibeCut AI folder and find the file called:

INSTALL.bat

Double-click it. A black window will open and start installing things automatically.

If Windows shows a blue "Windows protected your PC" popup, click "More info" then "Run anyway". This is normal for scripts downloaded from the internet.
If Node.js isn't installed, the script will try to install it for you. You may need to click "Yes" on a popup asking for permission. After Node.js installs, close the window and double-click INSTALL.bat again.
When you see "INSTALLATION COMPLETE!", you're done with this step.
3

🔑Add Your API Key

The AI features need an API key. Here's how to get one for free:

  1. Go to aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the key (it looks like a long string of letters and numbers)
  5. In the VibeCut folder, find the file called .env.local
  6. Right-click it → Open with → Notepad
  7. Replace your_gemini_api_key_here with your actual key
  8. Save the file (Ctrl + S) and close Notepad
Can't see the .env.local file? In File Explorer, click View at the top, then check "Hidden items". Files starting with a dot are hidden by default.
4

🚀Run the App!

Double-click this file to start VibeCut AI:

START.bat

A browser window will open automatically with the app. That's it — you're ready to go!

To stop the app, close both command prompt windows that appeared (the main one and the minimized "VibeCut Backend" in your taskbar).

Mac Setup

1

💻Open Terminal

We need to use Terminal (Mac's command line) for all the steps:

  1. Press Cmd + Space to open Spotlight Search
  2. Type Terminal
  3. Press Enter to open it

A window with a black (or white) background and a blinking cursor will appear.

2

📁Get the Files

Option A — Clone with Git (recommended):

Copy and paste this into Terminal, then press Enter:

git clone https://github.com/akamrt/Vibe-video-editing-.git && cd Vibe-video-editing-
If Terminal says "git" is not found, it will offer to install the Xcode Command Line Tools. Click Install, wait for it to finish, then try the command again.

Option B — From a copied folder:

If someone gave you the folder directly, type cd (with a space after it) then drag and drop the folder from Finder into Terminal, and press Enter.

3

📦Install Everything

Type these two commands, pressing Enter after each:

chmod +x INSTALL.sh START.sh
./INSTALL.sh

This will install everything automatically. It may ask for your Mac password — this is normal.

The installer will:

  1. Install Node.js (via Homebrew) if you don't have it
  2. Download all app dependencies
  3. Download yt-dlp and ffmpeg (for video processing)
  4. Build the Python AI tracker (if Python is available — optional)
  5. Create the API key config file
When you see "INSTALLATION COMPLETE!", you're done with this step.
4

🔑Add Your API Key

Same as Windows — get a free key from aistudio.google.com/apikey.

Then in Terminal, type:

nano .env.local

Replace your_gemini_api_key_here with your key.
Press Ctrl + O (to save), then Enter, then Ctrl + X (to exit).

Alternatively, you can open .env.local with TextEdit: open -a TextEdit .env.local
5

🚀Run the App!

In Terminal, type:

./START.sh

A browser window will open with the app. To stop it later, press Ctrl + C in Terminal.

🔄How to Get Updates

When there's a new version, updating takes about 30 seconds. Make sure the app is stopped first, then:

If you installed using Git:

  1. Open the VibeCut AI folder
  2. Click in the address bar at the top of File Explorer, type cmd, and press Enter
  3. A black command prompt window will open. Type this and press Enter:
git pull && INSTALL.bat

If you received the folder as a copy:

Ask the person who gave it to you for the updated folder, and replace your old one. Your .env.local file (with your API key) won't be in the new copy, so either copy it over from the old folder, or set it up again.

If you installed using Git (recommended method):

  1. Open Terminal (Cmd + Space, type "Terminal", press Enter)
  2. Go to your VibeCut folder. If you put it on the Desktop:
cd ~/Desktop/Vibe-video-editing-

Then run:

git pull && ./INSTALL.sh
That's it! git pull downloads the latest changes, and INSTALL.sh handles any new dependencies. Your API key and settings are kept.

If you received the folder as a copy:

Ask the person who gave it to you for the updated folder. Copy your .env.local file from the old folder to the new one so you don't have to set up your API key again.

Your API key, settings, and any saved projects are never affected by updates — only the app code gets updated.

🔧 Troubleshooting

The video won't download / "bot" error
YouTube may require browser cookies. Install the "Get cookies.txt" extension in Chrome, go to youtube.com, export cookies, and save the file as www.youtube.com_cookies.txt in the VibeCut folder.
Windows says "protected your PC" / blocks the script
This is Windows SmartScreen. Click "More info" then "Run anyway". This is normal for any script not from the Microsoft Store.
The page says "localhost refused to connect"
The server might not have started yet. Wait 10 seconds and refresh. If it still doesn't work, close everything and double-click START.bat again.
I can't find the .env.local file
Windows: In File Explorer, click View → check "Hidden items".
Mac: In Finder, press Cmd + Shift + . (period) to show hidden files.
"Port is already in use" error
Another app (or a previous VibeCut session) is using the port. Close all command prompt / terminal windows and try again. On Windows, you can also restart your computer.
AI features say "API key not set"
You need to edit the .env.local file and add your Gemini API key. See Step 3 above for instructions on getting a free key.
