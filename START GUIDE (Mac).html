If something goes wrong

🔲 Blank white page in browser

Wait 15 seconds and press ⌘ + R to refresh. The app might still be starting up.

If it stays blank: close the Terminal window, then double-click "Start Vibe.command" again. It will clean up and restart.

📄 Still opens as a text file

Open Terminal again (Step 1 above) and paste the same command from Step 3 again. This re-applies the fix.

❌ "command not found" error

The VibeCut folder was not found in Downloads, Documents, or Desktop. Move the folder to your Downloads folder, then paste the command again.

⏳ App is very slow to start

Normal on the first run — it is installing everything (can take 5 minutes). After the first time it will start in seconds.
